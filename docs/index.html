Installation

Follow these simple steps to get started:

1 Download or extract template files.
2 Open project folder in VS Code.
3 Open index.html in browser.

Customize Branding

To customize this template for your hosting company:

  1. Press CTRL + F
  2. Search:
YOUR-HOSTING-NAME

Replace it with your hosting brand name.

Example: YOUR-HOSTING-NAME → ExampleHost

Change Logo & Images

Replace logo and website images from:

/images/

Make sure to keep file names same or update paths in HTML.

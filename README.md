# 🚀 DhruvaHost - Hosting Website Template

A modern and responsive **hosting website template** designed for web hosting businesses, VPS providers, game hosting, cloud hosting, and server companies.

Built with clean HTML, CSS, and JavaScript for easy customization.

---

## 📂 Project Structure


```
docs/ → Documentation files
fonts/ → Fonts used in the template
images/ → Website images & assets
js/ → JavaScript files
index.html → Main homepage
products.html → Products/hosting page
```

## ⚙️ Customization Guide

## Change Hosting Name

To rename the hosting company name:

1. Open the project in your code editor.
2. Press:

```
CTRL + F
```

3. Search for:

```
YOUR-HOSTING-NAME
```

4. Replace it with your hosting brand name.
Example:
```
YOUR-HOSTING-NAME → DhruvaHost
```

## Change Logo

Replace the logo image inside:


```
images/
```


and update the path if required.

## Change Content

You can edit:
```
Hosting plans
Pricing
Features
Contact details
Company branding
```
directly from the HTML files.
